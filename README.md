# Simple ToDo App

シンプルなデスクトップ向けToDoアプリです。

## 特徴
- **シンプルなデザイン**: 直感的で使いやすい。
- **デスクトップ通知**: 設定した時間になると、Windowsの通知センターでお知らせ。
- **スマートな日時入力**: `flatpickr` を搭載し、期限設定が可能。
- **データの永続化**: アプリを閉じてもタスクは自動的に保存。
- **完了チェック機能**: タスクを完了状態にして管理。

## インストール方法
1. [Releases](https://github.com/LEON-72/todo-app/releases) ページにアクセスします。
2. 最新バージョンの `my-todo-app-Setup-1.1.0.exe` をダウンロードします。
3. ダウンロードしたファイルを実行してインストールしてください。

## 使用技術

- Electron
- Flatpickr
